# HotLot — Zero-Config Hosting (No Google Key)

1) Create a GitHub repo named **HotLot** and upload all files from this folder.
2) Go to https://vercel.com/new → Import **HotLot** → Deploy.
3) Share your live link (e.g. https://hotlot.vercel.app).

Tip: In the app, you can enter an origin like `-37.8183,144.9671` for exact CBD-based commute estimates.
